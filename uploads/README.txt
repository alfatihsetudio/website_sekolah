Folder ini digunakan untuk menyimpan file yang diupload.

Subfolder:
- materials/    : File materi pembelajaran
- assignments/  : File contoh tugas
- submissions/  : File submission tugas dari murid

Folder ini dilindungi dengan .htaccess untuk mencegah akses langsung.
File hanya bisa diakses melalui script download.php dengan permission check.

Pastikan folder ini memiliki permission write (755 atau 777).
